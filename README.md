# MES Project

스프링 부트 + JSP 기반 mes 프로젝트

## 💻 프로젝트 소개

mes 프로젝트 입니다.

## 🗓 개발 기간

* 22.04.26일 ~ 22.05.24일

## 👥 맴버구성

* **팀원1** - 로그인, 회원가입, ID찾기, PW찾기, 마이 페이지(예매 페이지), 홍보 및 평점관리, PPT제작, 발표
* **팀원2** - 메인 페이지, 배포 CSS
* **팀원3** - 1대다 문의 게시판(CRUD), 공지사항 게시판(CRUD)
* **팀원4** - 1대다 문의 게시판(CRUD), 공지사항 게시판(CRUD)
* **팀원5** - 로그인, 회원가입, ID찾기, PW찾기

## ⚙ 개발 환경

* Java 8
* JDK 1.6.8
* IDE : STS 3.9
* Framework : Springboot(2.x)
* Database : Oracle DB(11xe)
* ORM : Mybatis

## 🚀 주요 기능

* 로그인 · 성별분석 · WIKI 이동
* DB업 검증

## 📚 참고 자료

MES 개발 핵심 업무 영역
1. 기준정보 관리

품목 마스터 (원자재, 반제품, 완제품)
BOM(Bill of Materials) 관리
공정/라우팅 정보
설비/작업장 정보
작업자 정보

2. 생산계획 관리

작업지시 생성 및 배포
일/주/월 생산계획 수립
공정별 스케줄링

3. 생산실적 관리

작업 시작/종료 처리
생산수량 실적 등록
공정별 진행현황 추적
작업시간 관리

4. 품질관리

수입검사 (원자재)
공정검사
출하검사
불량유형 관리 및 분석

5. 재고/자재관리

원자재 입고/출고
재공재고(WIP) 관리
완제품 입고
LOT 추적

6. 설비관리

설비 가동현황 모니터링
비가동 사유 관리
설비 이력 관리

7. 모니터링/분석

생산현황 대시보드
실적 대비 분석
KPI 지표 (가동률, 불량률, 생산성 등)
