# SpringBoot-Project-MEGABOX

스프링 부트 + JSP 기반_영화 예매 사이트

## 💻 프로젝트 소개

MEGA BOX를 컨셉으로 만든 영화 예매 사이트입니다.

## 🗓 개발 기간

* 22.04.26일 ~ 22.05.24일

## 👥 맴버구성

* **팀원1** - 로그인, 회원가입, ID찾기, PW찾기, 마이 페이지(예매 페이지), 홍보 및 평점관리, PPT제작, 발표
* **팀원2** - 메인 페이지, 배포 CSS
* **팀원3** - 1대다 문의 게시판(CRUD), 공지사항 게시판(CRUD)
* **팀원4** - 1대다 문의 게시판(CRUD), 공지사항 게시판(CRUD)
* **팀원5** - 로그인, 회원가입, ID찾기, PW찾기

## ⚙ 개발 환경

* Java 8
* JDK 1.6.8
* IDE : STS 3.9
* Framework : Springboot(2.x)
* Database : Oracle DB(11xe)
* ORM : Mybatis

## 🚀 주요 기능

* 로그인 · 성별분석 · WIKI 이동
* DB업 검증
