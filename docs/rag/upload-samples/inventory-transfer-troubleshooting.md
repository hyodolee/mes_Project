# 자재 이동 장애 조치 문서 샘플

## 문서 목적
이 문서는 RAG 업로드 테스트용 재고/물류 문서입니다.
자재 이동 실패가 발생하면 출발 위치, 도착 위치, PLC 이벤트, 로케이션 재고 상태를 함께 확인합니다.

## 이동 실패 확인 순서
- 이동 상태가 FAILED이면 MCS 이동 관리에서 출발/도착 위치를 확인합니다.
- 도착 위치가 비어 있거나 잠금 상태이면 로케이션 재고 화면에서 사용 가능 여부를 확인합니다.
- PLC 이벤트에 목적지 누락 메시지가 있으면 태그 매핑표의 목적지 태그를 확인합니다.
- RAG 테스트 식별 문구: TRANSFER-RAG-LOGISTICS-TEST.

## 권장 조치
동일 구간에서 실패가 반복되면 경로 관리 화면에서 해당 경로의 우선순위와 사용 여부를 확인합니다.
