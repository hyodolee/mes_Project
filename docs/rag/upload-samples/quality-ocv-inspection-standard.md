# OCV 검사 기준서 샘플

## 문서 목적
이 문서는 RAG 업로드 테스트용 품질 문서입니다.
OCV 검사에서 전압 편차가 기준을 벗어나는 경우 검사 부적합으로 판단합니다.

## 판정 기준
- OCV 전압 편차가 기준 범위를 초과하면 검사 부적합으로 분류합니다.
- 동일 LOT에서 OCV 부적합이 반복되면 Formation 조건과 셀 안정화 시간을 함께 확인합니다.
- RAG 테스트 식별 문구: OCV-RAG-QUALITY-TEST.

## 권장 조치
OCV 부적합이 발생하면 검사 결과 화면에서 해당 LOT의 불합격 수량을 확인하고,
불량 이력 화면에서 동일 품목의 최근 불량 원인을 비교합니다.
