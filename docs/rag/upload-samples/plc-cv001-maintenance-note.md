# CV-001 설비 점검 노트 샘플

## 문서 목적
이 문서는 RAG 업로드 테스트용 PLC/설비 문서입니다.
CV-001 컨베이어에서 과부하 또는 인터록이 발생할 때 점검할 항목을 정리합니다.

## 점검 항목
- CV-001 모터 과부하가 발생하면 벨트 장력과 베어링 온도를 먼저 확인합니다.
- 인터록 신호가 유지되면 안전 커버 스위치와 비상정지 회로를 확인합니다.
- RAG 테스트 식별 문구: CV001-RAG-EQUIPMENT-TEST.

## 권장 조치
과부하가 반복되면 설비 비가동 이력에 정비 사유를 남기고, 재가동 전 무부하 테스트를 수행합니다.
